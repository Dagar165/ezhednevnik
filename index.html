Утреннее планирование
Понедельник — штаб. 20 минут, чтобы разложить неделю.
1. Разбери входящие
Каждую задачу — либо раскидай по дням, либо удали. Цель: входящие = 0.
2. Обнови зоны
Пройдись по зонам. Добавь новое, удали неактуальное.
3. Раскидай по неделе
Смотри свободные окна (зелёные). Не больше 2-3 задач на тяжёлые дни.
4. Главная задача дня
Выбери ОДНУ задачу на сегодня — самую важную. Она будет вверху «Сегодня».

Вт и Сб — тяжёлые. Не грузи сверх расписания.

Пн и Пт — дни для проектов (школа, Келек, волонтёры).

Чт утро — 3-4 часа глубокой работы до гимназии.

Вс — не трогай. День с женой.
